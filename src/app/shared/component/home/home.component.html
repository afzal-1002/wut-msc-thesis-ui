Welcome to Bug Estimation AI Tool

Enhancement of bug time estimation

What is This Project About?

This application is part of my Master's thesis titled: Enhancing Bug Resolution Time Estimation with AI in Spring Boot Microservices. The goal is to integrate AI models into a bug tracking system to predict how long it will take to resolve issues.

What Am I Going to Do?

  • Develop a microservices-based application using Spring Boot.
  • Integrate the Jira REST API to retrieve real-world bug and issue data.
  • Use AI models such as Gemini and DeepSeek to analyze historical bug data and estimate resolution time.
  • Automatically update Jira issue comments based on AI-generated analysis and recommendations.
  • Present AI predictions and insights through a user-friendly dashboard.
