Welcome to Bug Estimation AI Tool

Enhancement of bug time estimation

What is This Project About?

This application is part of my Master's thesis titled: Enhancing Bug Resolution Time Estimation with AI in Spring Boot Microservices. The goal is to integrate AI models into a bug tracking system to predict how long it will take to resolve issues.

What Am I Going to Do?

  • Build a microservices-based application using Spring Boot.
  • Integrate Jira API to fetch real bug data.
  • Train AI models (like ChatGPT or DeepSeek) on historical bugs to predict resolution time.
  • Visualize predictions through a user-friendly Angular dashboard.
  • Send alerts or notifications based on estimated resolution delays.
